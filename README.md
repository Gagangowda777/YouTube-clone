# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

## YouTube Clone
This is a learning project where i have built my first full-stack application. 
It is a clone of Youtube which uses MERN stack.

## link to project 
https://github.com/Gagangowda777/YouTube-clone

## Technologies used
- Frontend: React, React Router, Axios
- Backend: Node.js, Express.js, MongoDB
- Authentication: JWT (JSON Web Tokens)
- Database: MongoDB (MongoDB Atlas or local instance)
- Version Control: Git

## Features of this project
- user can register/login using their email, name and password 
- user can upload videos on their channel just like real youtube 
- user can also view others video 
- user can comment on video also they can edit or delete the comment 
- user can also edit or delete their uploaded video on there dashboard 
- user can toggle dark/light theme 
- user can search videos using search bar or filter buttons 
- user can like or dislike a video 
- user can watch video without ads! (that's a loophole for now)

## Learning from this project
- This project helped me to understand how a web application works end-to-end 
- I understood how the back-end communicates with front-end and understood the role of servers
- I learned how to use MongoDB and how to store and retrieve data from it (CRUD operations)
- I learned how to use JWT for authentication and authorization 

## How to run the application 
- step1 : clone/pull req. the project onto your local machine 
- step2 : Install node modules (npm install) 
- step3 : to run the app use -> npm run dev (for both frontend and backend)
- step4 : use http://localhost:5173/ to view the application
